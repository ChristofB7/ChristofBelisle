John Conway's Game of Life.

  • By Christof Belisle
  • May 19, 2021

John Conway's game of life is a way to simulate live and dead cells based on rules. Originally as a C++ assignment, I turned this assignment into a fun and interactive JavaScript program that plays in the browser*.

*soon to come

The Rules


                          Rule 1: Any live cell with less than 2 live neighbors dies.
                          Rule 2: Any live cell with 2 or 3 neighbors lives.
                          Rule 3: Any live cell with more than 3 neighbors dies.
                          Rule 4: Any dead cell with exactly 3 neighbors becomes a live cell.

The Start

As the most diffcult assignment of the introductory course at Bishop's University the Game of Life assignment became a challenge. It was not too difficult but I was eager to show off my progress to my friends at the dinning hall. I was proud to see the skills I had aquired in my first experience programming. It holds a special place in my heart, which is why it became the perfect project to recreate in JavaScript

Game of Life became an easy project to recreate in JavaScript with a better understanding of coding fundementals. Once created in JavaScript the next step was to get it to show on an html page.
