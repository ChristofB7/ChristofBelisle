Octopus Escape - Unity Based Game.

  • By Christof Belisle
  • May 27, 2021

Developing a game in 48 hours is a lot of work.

There are so many youtube videos of people developing games in 2 days and its inspiring. You can see the the entire project from start to finish. You can see them brainstorm, designing the moving parts as blocks, get frustrated that its not working as intended, and then see the incredible final transformation. These videos are glorious and inspiring. Doing it yourself is a vastly different experience. I wanted to take on this challenge, so I signed up for GMTK's 2020 Game Jam.

The theme was "Out of Control". This posed a difficult task and was hard to come up with an idea for it. After a long brainstorm session I came up with the idea that you don't control your character. The idea was that you would give the character commands and then he would execute those commands. Developing the game was a huge challenge and a long 48 hours. By the end I had created something that I was quite proud of. Pixel art was a big interest of mine and I hand drew all of the assets in Aseprite. GMTK Game Jam YouTube Video.

The version in this browser has had a lot of tweaking done to it. Some of the improvements include:

  • ignore the delay between level load and the first movement
  • make the first few levels easier
  • create a move intuitive tutorial
  • lessen the distance to the end
  • added levels select
  • added practice mode
A lot of these upgrades came from the comment section of our submission where others had played it.
